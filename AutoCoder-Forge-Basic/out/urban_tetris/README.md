# Urban Block Stack

An urban hip-hop themed Tetris game where you stack blocks to build skyscrapers and create beats.

## How to Play

- Use arrow keys or WASD to move.
- Press SPACE to jump.
- Collect items to increase your score.
- Avoid enemies to keep your lives.

## Features

- Urban hip-hop theme
- Platformer gameplay
- Score system
- Lives system

## Requirements

- Godot 4.1 or later

## Running the Game

1. Open the project in Godot.
2. Press F5 to run.
